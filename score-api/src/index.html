🚀 Scores API

Welcome to the Scores Backend

This API powers the Scores project, providing a comprehensive set of features for the frontend application.

  • Version: 1.0.0
  • Base API Path: /api
  • API Documentation: Swagger UI (/docs)
  • Authentication: JWT (Bearer Token)

📦 Key Modules

  • 🔐 Authentication: User registration, login, and profile management.
  • 🛍️ E-commerce: Products, shopping cart, orders, and user addresses.
  • 🍳 Recipes: Recipe management, favorites, ratings, comments, and shopping lists.
  • 🎮 Real-time: Live game scores streamed via Server-Sent Events (SSE).
  • 💬 Social: Chat threads and messaging.
  • 💪 Fitness: Available and finished exercises tracking.
  • 🤖 AI Bot: User intent detection.
  • 👥 Employees: Basic CRUD operations for employee records.

💡 Quick Tips for Frontend & Testers

  • Check the Swagger Docs to explore and test all endpoints directly from your browser.
  • Protected endpoints require an Authorization: Bearer <token> header. Obtain a token via the /api/auth/login endpoint.
  • Uploaded media (like recipe images) are served statically from the /uploads/ directory.
  • The games scores stream can be accessed at /api/games/scores.
